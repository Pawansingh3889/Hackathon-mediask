# HealthQ — Medical Q&A Platform (Hackathon Project)

A healthcare question-and-answer web platform built during a team hackathon. Users can browse medical articles by category, ask health-related questions, and filter community responses by condition type.

This is my only collaborative project on GitHub — 5 contributors, 45+ commits, real Git workflow experience.

---

## Features

- **Medical articles library** — browse articles on Allergies, Alzheimer's, Cancer, Eye Health, Hepatitis C, Memory, and more
- - **Community Q&A** — submit and browse health questions with likes, views, and star ratings
  - - **Category filtering** — JavaScript-powered dynamic filter across 13 medical categories (Mental Illness, Surgery, Cancer, Cardiovascular, Infection, etc.)
    - - **User authentication** — login page with form handling
      - - **Responsive layout** — CSS custom properties and flexible box model
        - - **Bookmarking** — save questions for later reference
